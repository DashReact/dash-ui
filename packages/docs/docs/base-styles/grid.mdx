# Grids

Utilities to control the grid of the elements

## Basic

| Class    | Properties                                                             |
| -------- | ---------------------------------------------------------------------- |
| **grid** | `display: grid;grid-template-columns: repeat(12, 1fr);grid-gap: 1rem;` |
