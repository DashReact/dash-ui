# Themes

### Light

It is the default theme without any variation

### Dark

For the dark theme, two variables of the main theme are simply overwritten

| Name                | value                  |
| ------------------- | ---------------------- |
| _--dash-font-color_ | <Color>#e1e1ff</Color> |
| _--dash-bg-color_   | <Color>#252525</Color> |
